About The Project

Learn how our pest detection system works

How It Works

The Pest Detection System uses advanced machine learning techniques to analyze audio recordings and identify various pest species based on their unique sound signatures.

Key Features

  • Real-time Audio Analysis: Process and analyze audio files in real-time
  • Multi-species Detection: Identify multiple pest species including Caribbean fruit fly, fire ants, termites, and various beetles
  • Detailed Information: Comprehensive details about each detected pest
  • Visualizations: Audio waveform and spectrogram displays
  • Advanced Analysis: Frequency analysis and confidence scoring

Technical Details

The system uses a convolutional neural network (CNN) trained on a dataset of pest sounds. Audio is processed as follows:

  1. Audio is captured at 16kHz sample rate
  2. Recordings are converted to mel-spectrograms
  3. A CNN model analyzes the spectrograms to identify pest species
  4. Multiple predictions are combined for more accurate results
  5. Frequency analysis verifies if dominant frequencies match expected pest ranges

Detected Pest Species

Our system can detect the following pest species based on their acoustic signatures:

  • Asian tiger mosquito
  • Caribbean fruit fly
  • Fire ants
  • Termites
  • Asian longhorned beetle
  • Black vine weevil
  • Mediterranean fruit fly
  • Butterfly
  • June beetle
  • Queensland fruit fly

Usage Instructions

Using the Pest Detection System is simple:

  1. Record audio using the built-in recorder or upload a WAV file
  2. Click "Analyze Audio" to process the sound
  3. Review the detection results, including confidence scores and pest information
  4. For best results, record clear audio with minimal background noise
