Configuration

To deploy this system with your own model files:

  1. Place your pest_detection_model.h5 in the models/ directory
  2. Place your class_names.npy in the models/ directory
  3. Alternatively, set the MODEL_URL and CLASS_NAMES_URL environment variables to download them automatically

Deployment Instructions

This system can be easily deployed to various hosting platforms:

Local Deployment
python app.py
Docker Deployment
docker build -t pest-detection .
docker run -p 5000:5000 pest-detection
Heroku Deployment
heroku login
heroku create pest-detection-app
git push heroku main

For more detailed deployment instructions, please refer to the documentation.

Environment Variables

The following environment variables can be used to configure the application:

Variable Description
PORT Port on which the server listens (default: 5000)
MODEL_PATH Path to the model file (default: models/pest_detection_model.h5)
CLASS_NAMES_PATH Path to the class names file (default: models/class_names.npy)
MODEL_URL URL to download the model from
CLASS_NAMES_URL URL to download the class names from
